Caracteristicas Elementos/Técnicas Exemplos
Layout Foi utilizado o CSS Flexbox na construção das páginas. o CSS Grid também está presente em algumas. Posição de todos os elementos do website.
Responsividade Foram utilizadas media queries que adaptam o layout para o telemóvel ou o PC, conforme o tamanho da janela. Todas as media queries do website.
Seletores Foram usados diversos tipos de seletores CSS no website. .class, :nth-child(n), :hover, .class1.class2, element, element, entre outros
Etiquetas Foram usadas quase todas as etiquetas sugeridas, nas diversas páginas. header, main, footer, section, aside, figure, svg, table, thead, tbody, tr, td, entre outras.
Animações Foram feitas diversas animações, aplicadas em todas as páginas do nosso website. Animação do aparecimento dos títulos das páginas, do preenchimento dos mesmos e o efeito de fade ao carregar a página.
